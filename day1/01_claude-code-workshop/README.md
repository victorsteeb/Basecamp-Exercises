# Claude Code Workshop

Use Claude Code to explore, modify, and extend a live factory inventory management system — a real Vue 3 frontend and FastAPI backend. The exercise is open-ended: there's no single right output. The point is to experience Claude Code as a development partner on a codebase you didn't write.

## Step-by-step guide

- **Live site (preferred):** https://claude-code-workshop.netlify.app/ — enter any name and any workshop code
- **Offline backup:** If the companion site is unavailable, open [`workshop-guide.pdf`](./workshop-guide.pdf) in this folder for the full step-by-step instructions.

The exercise has 12 core steps plus an Expert Challenge.
