Key Patterns

Data Loading (every view)

1. loading.value = true
2. await api.method(filters)
3. data.value = response
4. catch → error.value = msg
5. finally → loading.value = false
Template shows: loading spinner → error message → data

Reactivity Model

ref() — mutable raw data
computed() — derived/filtered views
watch() — trigger reloads on filter change
onMounted() — initial load
Raw API data lives in refs; computed properties derive charts, tables, KPIs from them.

Backend Filtering

apply_filters() — warehouse, category, status
filter_by_month() — YYYY-MM or Qn-YYYY
All data held in Python lists in memory. No ORM. Filtered using list comprehensions at request time. Inventory does not support month filter.
Warehouses: San Francisco · London · Tokyo  |  Categories: Circuit Boards · Sensors · Actuators · Controllers · Power Supplies  |  Data range: Jan–Dec 2025
