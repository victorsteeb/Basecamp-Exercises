API Endpoints
GET /api/inventory All inventory items
warehousecategory
GET /api/inventory/{id} Single inventory item by ID
GET /api/orders All orders
warehousecategorystatusmonth
GET /api/orders/{id} Single order by ID
GET /api/dashboard/summary KPI summary — inventory value, low stock, pending orders, backlog count
warehousecategorystatusmonth
GET /api/demand Demand forecasts — current vs forecasted per SKU
GET /api/backlog Backlog items with purchase order status flag
GET /api/spending/summary Spending totals — procurement, ops, labor
GET /api/spending/monthly Month-by-month spending breakdown
GET /api/spending/categories Spending grouped by category
GET /api/spending/transactions Recent transaction history
GET /api/reports/quarterly Q1–Q4 stats — orders, revenue, fulfillment rate
GET /api/reports/monthly-trends Month-over-month order count and revenue
GET /api/tasks Task list
POST /api/tasks Create a task
DEL /api/tasks/{id} Delete a task
PATCH /api/tasks/{id} Toggle task completion
POST /api/purchase-orders Create a purchase order linked to a backlog item
GET /api/purchase-orders/{backlog_id} Get purchase order by backlog item ID
